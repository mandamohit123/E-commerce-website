Online Shopping System (Java)

A simple console-based Online Shopping System developed in Java that allows users to browse products, manage a shopping cart, and calculate total purchase cost.

Features
📦 View available products
➕ Add products to cart
➖ Remove products from cart
🛍️ View cart items
💰 Calculate total price
👤 User-based session (customer input)

Concepts Used

This project demonstrates core Java concepts:

Object-Oriented Programming (OOP)
Classes & Objects
Encapsulation
Collections (ArrayList)
User Input Handling (Scanner)
Control Structures (loops, switch-case)

Requirements
Java JDK 8 or above
Any IDE (IntelliJ, Eclipse, VS Code) OR terminal

How It Works
User enters their name
Menu is displayed with options:
View products
Add to cart
Remove from cart
View cart
Calculate total
User interacts through numeric input
System updates cart dynamically
